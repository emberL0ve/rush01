# rush01
これは 42 セッションの poolrush01 で、私と私のバンドによって直接作成されました。
 
 コードをコンパイルします。:

 ```$ gcc -Wall -Wextra -Werror -o rush01 check.c rush01.c solve.c```

エグゼキュータの後にパラメータを追加します。:

  ```$ ./rush01.exe "4 3 2 1 1 2 2 2 4 3 2 1 1 2 2 2"```

これがターミナルに表示された結果です :
 ```
$ ./rush01.exe "4 3 2 1 1 2 2 2 4 3 2 1 1 2 2 2"
1 2 3 4
2 3 4 1
3 4 1 2
4 1 2 3
 ```
